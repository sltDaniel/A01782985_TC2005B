border

border-left

Se podría decir que es una abreviatura para las propiedades como:

  • ancho del borde izquierdo
  • el estilo
  • color del borde

Podemos observar como al hacer uso de border-left se genera un solo borde del lado izquierdo. En este ejemplo utilizamos el double.

border-right

Es igual una abreciación para:

  • ancho del borde derecho

  • color
