Cheat Sheet de HTML

a - Hyperlinks

Sirven para hacer un link de una página a otro recurso en la web, sea otro archivo en el servidor web, u otra página web.

Por ejemplo: Si le da click aquí lo llevara a la página de de Google

p - Paragraph

Sirve para poner párrafos de texto, y que se puedan visualizar en la página web.

Un ejemplo de esto es lo que se está viendo en este momento.

h1/h6 - Header

Sirve para poner encabezados dentro de la página web. Empieza en h1 y temina en h6, esto define el tamaño que el encabezado va a tener. En pocas palabras es para establecer niveles a los encabezados dentro de una página.

Por ejemplo:

Encabezado princiapal (h1)

Encabezado de nivel 2 (h2)

Encabezado de nivel 3 (h3)

Encabezado de nivel 4 (h4)

Encabezado de nivel 5 (h5)
Encabezado de nivel 6 (h6)

div - Division

Se usa para crear secciones o agrupar contenido

Ejemplo:

Ejemplo de div

Esto es un párrafo dentro de un div

span

Este es un contenedor en línea. Esto quiere decir que se usa para poder aplicar estilo al texto o agrupar elementos en línea.

Ejemplo: Empezamos a escribir y ahora hacemos uso de span

i - Italic

Este se utiliza para poder hacer que una palabra o letra se vuelva cursiva. Se usa para cambiar el tono de voz o tono alternativo

Para poder utilizarlo se pone al principio de una letra o palabra.

Por ejemplo: La tarea estuvo fácil

u

Mostrar el texto subrayado

Ejemplo: Normal Subrayado

b - Bold

La b se utiliza para hacer que una palabra resalte, o en otras palabras negrillas. Para poder hacer uso de este lo que se debe de hacer es ponerlo justo antes de una palabra.

Por ejemplo: Mi nombre es Daniel

strike

Sirve para poder tachar el texto. Subraya el texto con una línea horizontal por el medio.

Por ejemplo: Normal Texto tachado.

small

Usamos small para poder hacer el texto más chico. Pasamos de grande a mediano o de pequeño a más pequeño.

Por ejemplo: Texto más pequeño.

strong - Strong

Se utiliza cuando queremos que una palabra tenga importancia. Del mismo modo que b, este pone en negrillas la palabra.

Por ejemplo: Advertencia: tengo que hacer la tarea.

em - Italic (cursiva)

Es similar cuando se usa la i, pero la diferencia es que está se utiliza para hacer énfasis en el texto

Depende de tu texto es si vás a utilizar "i" o "em", su singificado semántico

Por ejemplo: México es el mejor país.

articel - Artículo

Representa una composición auto-contenida en un documento, página, sitio, etc. Que se quiere que sea distribuíble, reutilizable de manera independiente. Un ejemplo de esto se podría ver como, artículo de revista o periódico, widget, comentario, etc.

Por ejemplo:

aside - Aside

Este elemento representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento Son representadas como barras laterales o interseciones.

Por ejemplo: Pueden ser, explicaciones al margen, publicidad, en páginas web información del perfil o enlaces a blogs relacionados.

footer - Pie de página

Para representar un pie de página. Aquí normalmente se encuentra información del autor, derechos de autor, enlaces o documentos relacionados.

Ejemplo:

©copyrights

nav - Navegation

Sección de los principales enlaces de navegación en una página. Este debe reservarse solo para secciones de navegación primaria, como navegación global, tabal de contenidos, enlaces anteriores/siguientes, u otros grupos de enlaces de navegación.

Se vinculará a otras páginas dentro del mismo sitio web o a partes de la misma página web.

form

table

Poder representar información de manera de tabla. Los datos están organizados en columnas y filas haciendo uso del tag "tr" y "td" dentro de table. tr se usa para poder agregar una fila. td se utiliza para agregar una columna.

Por ejemplo:

Column 1 Column 2 Column 3
Column 1 Column 2 Column 3

ol - Order List

Sirve para crear una lista ordenada, esta lista puede ser tanto numérica como alfabética. El tag "li" es para definir cada elemento de la lista

Por ejemplo: La primera es numérica y la seguna es alfabética.

  1. Leche
  2. Pan

  2. Pan
